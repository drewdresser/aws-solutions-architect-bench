Scoring Methodology

Practice Exam 34%

AWS certification-style multiple-choice questions covering core services, architecture patterns, and best practices. Binary scoring: correct or incorrect.

Architecture Design 33%

Diagram understanding and architectural reasoning tasks. Models analyze AWS architectures, identify services, trace data flows, and assess risks. Rubric-based scoring.

CDK Synthesis 33%

Infrastructure-as-code generation using AWS CDK (Python). Code must successfully synthesize to CloudFormation. Binary scoring: synth passes or fails.

Overall Score Formula

Overall = 0.34 × practice_exam + 0.33 × architecture_design + 0.33 × cdk_synth

Score Confidence

Scores may vary between runs. Expected variance by category:

  • Practice Exam: ±5% (binary scoring, 50 items)
  • Architecture Design: ±10% (rubric-based scoring with LLM judge)
  • CDK Synthesis: ±5% (binary pass/fail, 40 items)

When models score within 5% of each other, consider them statistically equivalent.

Reproducibility

To reproduce results:

  1. Clone the repository
  2. Run uv sync && make bench && make board.json
